#### 🌟 Featured Work

**GraPLUS** — *Graph-based Placement Using Semantics for Image Composition*
> Published thesis (CVIU, Elsevier, Q1). Scene graphs + LLM embeddings for semantic object placement, 92.1% accuracy, 8.3% absolute improvement over prior SOTA.
> [View Repository](https://github.com/mamadkhaleghi/GraPLUS)

**SemanticDiffuse** — *Learned Harmonization for Semantic Object Placement*
> Sequel to GraPLUS. Integrates pretrained harmonization models (PCTNet/libcom) onto GraPLUS's placement pipeline; designed a reference-set-aware dual-FID + no-reference harmony-score protocol after identifying that OPA's own composite images are themselves unharmonized.
> [View Repository](https://github.com/mamadkhaleghi/semantic-diffuse)

**Relational-Grounding-Bench** — *Does Relational Context Improve VLM Grounding?*
> Benchmarks whether Visual Genome relation triplets improve Qwen2.5-VL-3B's bounding-box grounding on RefCOCO, across zero-shot, relation-prompted, and QLoRA fine-tuned conditions, stratified by relational/positional/attribute expression type. Finding: QLoRA fine-tuning closes ~16% of the relational-accuracy gap; explicit relation context (prompted or trained) does not improve on it.
> [View Repository](https://github.com/mamadkhaleghi/relational-grounding-bench)

**ESC-50 Audio Classification via Vision Transformers**
> Spectrogram-as-image transfer learning: fine-tuned ViT vs. from-scratch CNN on identical stratified split — 88% vs 44.3% accuracy.
> [View Repository](https://github.com/mamadkhaleghi/audio_vision_classifier)
